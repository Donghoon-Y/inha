# Numerical Analysis (with MATLAB)

수치해석 수업 과제 모음. MATLAB Live Script(`.mlx`) 및 Jupyter Notebook(`.ipynb`)으로 작성.

---

## 📄 과제 목록

| 파일 | 주제 |
|------|------|
| `Assignment_01_Accuracy` | 수의 정확도와 오차 — Taylor 급수 기반 절대·참·근사 오차 계산 및 시각화 |
| `Assignment_02_Root_Finding1` | 근 탐색법 1 — 이분법(Bisection), 고정점 반복법(Fixed-Point) |
| `Assignment_03_Root_Finding2` | 근 탐색법 2 — Newton-Raphson, Secant Method |
| `Assignment_04_LinearSystems` | 선형 연립방정식 — Gauss 소거법, LU 분해 |
| `Assignment_05_Reg_Int` | 회귀분석 및 적분 — 최소자승법, Trapezoidal Rule |
| `Assignment_06_Integration_Differentiation` | 수치 적분·미분 — Simpson 공식, 수치 미분 |
| `Assignment_07_IVP_BVP` | 초기값·경계값 문제 — Runge-Kutta 4차법, Shooting Method |
| `FinalExam_1216` | 기말고사 종합 풀이 |

---
